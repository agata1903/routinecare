# Routine Care API

API REST desenvolvida em Java com Spring Boot para gerenciamento de:

- Medicamentos
- Hidratação diária
- Registro de tomadas

## Objetivo

O objetivo do projeto é auxiliar usuários no acompanhamento da rotina de autocuidado, permitindo registrar medicamentos tomados e consumo diário de água.

## Tecnologias usadas

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Maven
- Postman para testes

## Funcionalidades

### Usuários
- Cadastro de usuário
- Busca de usuário
- Atualização de registro de usuário
- Remoção de usuário

### Medicamentos
- Cadastro de medicamentos
- Listagem de medicamentos por usuário
- Registro de medicamento tomado
- Atualização de registro de medicamento tomado
- Remoção de registro

### Hidratação
- Registro de consumo de água
- Consulta do total diário consumido
- Atualização de registro
